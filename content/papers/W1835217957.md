## Abstract

PBL (Problem Based Learning) e uma estrategia educacional centrada no aluno, cuja apresentacao dos problemas antecede a exposicao de conteudos. Problemas sao, portanto, um dos principais elementos do PBL. Este artigo descreve o PBL Manager, um sistema WEB desenvolvido para oferecer suporte a elaboracao e compartilhamento de problemas. Sao apresentados o contexto para o surgimento do PBL Manager e as caracteristicas da ferramenta. Finalmente, uma discussao sobre os aspectos positivos e a evolucao do PBL Manager sao considerados.
