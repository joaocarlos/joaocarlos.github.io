## Project Overview

The main objective of this project is to develop an action aimed at the popularization of science in the communities of Chapada Diamantina, through the development of an Internet application tool, entirely based on open source code and on a free operating system platform (GNU/Linux), designed to promote this region through a database that covers its various aspects, such as: cultural, socio-economic, historical, geographical, tourist and leisure.

## Objectives

1. Develop web platform based on open source
2. Create comprehensive regional database
3. Promote science popularization
4. Support regional development

## Aspects Covered

-   Cultural
-   Socio-economic
-   Historical
-   Geographical
-   Tourist
-   Leisure
